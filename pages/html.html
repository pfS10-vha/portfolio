Compétence Technique - HTML

Définition

    HTML est un langage de balisage web, c'est-à-dire qu'il va permettre de mettre du texte, des liens ou encore des images sur une page internet. C'est un langage principalement destiné aux développeurs web afin de réaliser des sites internet, mais il peut aussi être utilisé par tout autre corps de métiers qui souhaiterait présenter ses travaux à l'aide d'un site internet. Cela peut aller d'un administrateur système qui souhaiterait rendre compte de données sous la forme d'un tableau sur une page web plutôt que d'utiliser un fichier Excel, d'un artiste souhaitant partager ses créations sur son propre site web ou d'un professionnel voulant rendre son portfolio accessible en ligne. C'est grâce à ces multiples possibilités qu'offre le langage HTML qu'il est devenu très prisé pour la création de sites web, ainsi que la base de beaucoup d'autres langages web, y compris dans le milieu professionnel.

Éléments de preuve

    Lors de ma première année d'études, j'ai eu l'opportunité de réaliser un site web pour une petite boutique de ventes de vêtements. L'objectif était d'exposer, en ligne, les différents vêtements disponibles dans la boutique en fonction des saisons. De plus, le site devait être le plus simple possible, de sorte que le site soit facile à déployer et que les commanditaires puissent facilement modifier le texte contenu dans le code lorsqu'ils le souhaitent. Pour ces raisons, nous avons opté pour deux langages très simples d'utilisation, HTML et CSS. HTML permet d'afficher le texte sur un site, tandis que le langage CSS permet de changer la position du texte sur la page ou sa couleur. Cela a permis aux commanditaires de facilement déployer le site internet et de pouvoir modifier le texte et les images lorsqu'ils le souhaitaient.

    Il m'est aussi arrivé à plusieurs reprises d'avoir recours au langage HTML. Lors de mon alternance par exemple, j'ai très souvent l'occasion de réaliser des scripts qui font des tâches de manière automatique. Toutefois, si certains vont uniquement réaliser la tâche de façon journalière, d'autres vont devoir générer un rapport que nous pourrons consulter. Le souci étant qu'un script n'a pas pour objectif d'être esthétique et va, par conséquent, renvoyer un résultat peu lisible sous forme de texte. Pour pallier cela, pour plusieurs de mes scripts, j'ai fait en sorte de générer une page web HTML montrant les résultats sous forme de tableau. Aujourd'hui, cela nous permet d'avoir un rapport quotidien du résultat de ces scripts.

Autocritique

    Par ces nombreuses fois où j'ai eu l'occasion d'utiliser le langage HTML, je pense pouvoir prétendre avoir un très bon niveau dans cette compétence. Un point sur lequel je pourrais cependant m'améliorer serait la sécurité. En effet, j'ai souvent travaillé sur des petits sites web, des pages permettant d'afficher un rapport ou un tableau contenant des informations non sensibles et n'ai jamais réellement eu besoin de sécuriser ces pages. Il me serait donc bénéfique, rien que pour mon enrichissement personnel, d'apprendre à sécuriser une page web et peut-être même de réaliser un portail de connexion par exemple.

    En fin de compte, je recommande cette compétence non seulement à des personnes intéressées par l'informatique, mais également à quiconque souhaitant exposer ses travaux en ligne. C'est un langage assez simple à maîtriser et qui, dans le cas d'une candidature, donne un avantage considérable par rapport à un autre candidat qui n'aurait pas de site web pour présenter ses travaux.

Évolution

    Enfin, mon objectif principal serait principalement d'améliorer mes compétences en sécurité web, telles que la prévention des attaques XSS par exemple. Cela est très important puisque, de la même façon qu'un téléphone posé sur un banc toute une journée est amené à se faire voler, un site web non sécurisé et exposé sur internet est amené à se faire pirater un jour ou l'autre. Il est donc essentiel d'apprendre à sécuriser un site internet, et cela passe par quelques connaissances en cybersécurité. À cet effet, je prévois de suivre une formation sur la sécurité web ainsi que d'apprendre à l'aide d'une plateforme en ligne nommée HackTheBox.
